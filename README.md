# 🤖 AI Resume Updater — Telegram Bot

A Telegram bot that uses AI to update your resume based on any Job Description (JD).

## ✨ Features

- 🔍 **Analyze Match** — See how well your resume matches a JD
- 📄 **Update Resume** — AI rewrites your resume to match the JD
- 📝 **Cover Letter** — Auto-generate a professional cover letter
- 🎯 **Interview Questions** — Get likely interview questions with answer hints
- 📎 **PDF & Word Output** — Download updated resume as PDF or Word file

## 🛠️ Tech Stack

- **Python** — Main language
- **python-telegram-bot** — Telegram bot framework
- **Groq / Gemini / OpenRouter / Cohere / HuggingFace** — AI APIs (with auto-fallback)
- **ReportLab** — PDF generation
- **pdfplumber** — PDF text extraction
